Welcome to Flux Harmonic

Flux Harmonic is a "creative label" for the various projects of David Wilson.

Watch Live!

The work on these projects was performed live on YouTube and Twitch every Tuesday and Thursday from 3PM to 5PM UTC.

NOTE: The regular live streams are now on hiatus, see this page for more details.

Notes from all previous streams are also available.

Projects

  • Mesche: A Scheme-inpsired language designed for integration with C applications.
  • Substratic Engine: A game engine written in Mesche and C.
  • Syphon Break: A space shooter game written with Mesche and Substratic Engine.
  • Crash The Stack: A hacking-themed tile matching game written with Mesche and Substratic Engine.
  • Flux Compose: A creative composition studio for hackers.
