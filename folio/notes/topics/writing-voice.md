# Writing Voice

How Flux Harmonic writing should sound, for me and for any agent writing on my
behalf. The goal is simple: it should read like a person who's excited about the
work, not like a content team or an AI.

## The feel

- Conversational, but I don't dumb things down. The audience is smart and curious.
- Honest and direct. If something's hard, say it's hard. If there's a tradeoff,
  name it. No hype, no overselling.
- Enthusiastic where it's real. I use exclamation points at genuine moments of
  excitement! Not on every line, and never stacked.
- First person. "I'm building...", "we'll figure out...", "you might be wondering...".

## Rules that matter

- **No em-dashes.** Use a colon, a comma, parentheses, or just start a new
  sentence. Em-dashes are the clearest tell that something was written by an AI.
- **Be specific.** "Learn a lot" says nothing. "Rewrite the bullet engine to run
  off the music" says something. If I can't be specific, I cut the claim.
- **Plain words.** No "leverage", "utilize", "empower", "unlock the power of",
  "take it to the next level". Just say the thing.
- Contractions are natural. Stiff, contraction-free prose doesn't sound like me.

## The test

Read it back. If it sounds like a person talking, it's right. If it sounds like a
landing page or a manual, rewrite it.
