# A New Emacs in 5 Days

<!-- intro -->

## News

**Slate, a new Emacs, started five days ago and now has 163 commits:**

- First commit was Sunday July 12th at 20:54. It's ~19,000 lines with 3,625 lines of tests across 20 suites.
- Commits per day: 7, 17, 24, 28, 46, 41. It sped up.
- Editor: virtualized rendering, syntax highlighting, rainbow parens, show-paren, folding, undo/redo, auto-pairs, structural indent.
- Terminal: a native VT/ANSI core, a real pty, scrollback, mouse modes, bracketed paste, OSC-52 clipboard.
- Folder browser, find-file over real local files, 48 themes, on-demand translucency, and a mobile accessory bar that can enter Emacs chords with no Ctrl key.
- A live web app running in a pane, keeping its state across renders.

**Three more pieces got built alongside it:**

- sigil-vt, the terminal emulator core: **one day old**, 7 commits.
- Lantern, the desktop wrapper: **one day old**, 17 commits. GTK4 and WebKitGTK bound through dynamic FFI. No Rust, no Node, no C glue.
- sigil-system, capability-based file access for local and remote machines the way TRAMP does it: **two days old**.

**Releases this week:**

- sigil 0.17.16: DOM primitives for in-place reconciliation, and a hook that lets an app ship its own native module into a browser build.
- sigil-vt 0.1.0, then 0.1.1: the native terminal core, then a security patch for it.
- sigil-web-client 0.15.2: keyed reconciliation, so a focused element survives a re-render.
- sigil-system 0.1.1: canonical listing order and `fs-home`.

**Today alone, Slate took 12 merges:**

- Slate hung at 100% CPU trying to open its own release notes. One em-dash defeated an ASCII fast path and turned a whole-buffer scan quadratic. It now opens in five seconds.
- The terminal moved to the native core. `ls -R /gnu/store` renders as fast as you'd want to read it.
- Terminal scrollback got 16x faster, 143ms to 8.7ms a frame, and the live path got faster too instead of being traded away.
- The terminal now enforces its cell grid, so the Claude Code spinner stopped bouncing the line.
- A real heap out-of-bounds read, reachable by widening a terminal after scrolling, found and patched.
- The 5,000,000 iteration security gate that found it turned out to have no sanitizer linked into it at all. It had been reporting green its entire life. It now refuses to run unless it can prove it catches a planted bug.
- Shift+Tab sends a real backtab. The shift flag was never threaded into the key encoder at all.

## The Plan

- Build a which-key panel: press a prefix, then `?`
- Add a `*scratch*` buffer and `C-x C-e`
- Look at the parts that are still rough, and there are plenty
